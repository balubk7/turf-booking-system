# turf-booking-system
A console-based Smart Turf Booking System developed using Core Java, JDBC, and MySQL. The application allows users to register, log in, book available turf slots, manage bookings, and provides an admin module for managing users and bookings. The project demonstrates object-oriented programming, database connectivity, and backend application development.

## Technologies Used
- Core Java
- JDBC
- MySQL
- Object-Oriented Programming (OOP)
- Git & GitHub

## Features
- User Registration & Login
- Admin Login
- Turf Slot Booking
- Booking History
- Input Validation
- MySQL Database Integration
- Modular Java Classes
